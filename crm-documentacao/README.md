## Documentação do Projeto Integrador — CRM Abgail

Este repositório centraliza todos os documentos produzidos durante o desenvolvimento do Projeto Integrador.  
Aqui estão reunidos os arquivos referentes ao banco de dados, backend, frontend e ao escopo geral do projeto.

A organização desta pasta foi planejada para acompanhar a evolução do projeto, permitindo atualizações contínuas e facilitando a leitura por parte dos avaliadores.

A seguir, está apresentado o esqueleto da estrutura de diretórios e seus conteúdos.

> ⚠️ Nota: Esta documentação é evolutiva e será atualizada conforme as etapas do Projeto Integrador avançam. Este README descreve apenas a estrutura geral deste repositório.
